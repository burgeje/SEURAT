SEURAT Architecture

SEURAT Architecture was originally designed to keep track of the architectural decisions made during the software development process. The features are now integrated with SEURAT baseline on version 6.0.0. SEURAT Architecture is currently an early prototype, as more research and development efforts are needed before it has all the designated functionalities.

 

Currently, SEURAT Architecture is integrated with SEURAT in three ways: it has its own “Pattern Library” that keeps track of the available patterns and their data, a “XML Import/Export” tool that will import and export the pattern library and the rationale ontology, and a “Generate Candidate Pattern” wizard which generates patterns as alternatives in a decision within the rationale explorer.
