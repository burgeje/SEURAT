UML Association – Requirements

To use the current version of SEURAT with your UML model, all of the following requirements must be satisfied.

·      You must have installed EMF, UML2 bundles with Eclipse, as well as Eclipse Platform SDK. (You may download Eclipse Modeling Tools, then install the SDKs in Help à Install New Software…)

·      The UML model of your design must be UML2 compatible. Older versions of argouml use XMI 1.x and cannot be used with SEURAT. For a complete list of compatible tools, please visit http://wiki.eclipse.org/MDT-UML2-Tool-Compatibility

·      The UML model must be stored locally, and you must have R/W permissions.

·      Additionally, if you would like to compile it with Papyrus, you must have Eclipse 3.5.x at the time this manual is created. This is due to several packages not available to newer versions of eclipse yet.
