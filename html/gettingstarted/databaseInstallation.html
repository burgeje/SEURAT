Database Installation

In addition to the SEURAT plugin, you will need to install the following:

Eclipse (we are using 3.2.1)
Java (1.5.x)
MySQL 5.0

You will also need to have the database tables set up with some initial rationale. We have created a zip file with a database containing empty tables to store the rationale and with an initialized argument ontology. This file is available from the SEURAT download directory (http://www.users.muohio.edu/burgeje/SEURAT/Downloads.html). Once you are running SEURAT, you can set the preferences to point to the appropriate database (in this case, named SEURAT_Base) and to provide the username and password you wish to use with your MySQL installation. If you are using an older version of MySQL (not 5) you will need to use an account that does not have a password. Otherwise, any account with CREATE, DELETE, INSERT, SELECT, UPDATE privileges will work.
