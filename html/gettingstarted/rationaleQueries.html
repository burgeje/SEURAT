Rationale Queries

After installing SEURAT you will find a new tab in the Eclipse Search window. This tab provides an interface for querying the rationale using a number of different criteria. The Eclipse Search window can be found on the Eclipse menu bar using Search -> Search, and can also be accessed with the keyboard shortcut Ctrl-H. From the following pages you can get more information on using these queries with your project's rationale:

  • Finding Rationale Entities
  • Finding Common Arguments
  • Finding Requirements
  • Finding Status Overrides
  • Finding Importance Overrides
