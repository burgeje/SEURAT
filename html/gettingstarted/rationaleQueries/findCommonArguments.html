Finding Common Arguments

SEURAT provides a search query designed to locate arguments based on their frequency. This can be done for each of the following argument types: assumption, claim, and ontology entry.

  • Through the Eclipse search window navigate to the SEURAT Search Page and select the 'Common Argument' radio button.
  • Select which element type to search for among the claim, assumption and ontology entry options.
  • The checkbox labeled 'Find Selected Alternatives Only' will restrict the search to only those arguments which are used in selected, otherwise referred to as adopted, alternatives.
  • Use the Search button at the bottom right of the Eclipse Search Window to execute your query.

After the search completes, the applicable arguments are displayed in in the Eclipse search view. You can sort and browse the arguments using the total number of references, the number of times used to argue in support of an alternative, and the number of times it opposed an alternative.
