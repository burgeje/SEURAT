Finding Requirements

SEURAT provides a query that finds requirements with a specific status. This makes it possible to retrieve lists of requirements such as those that have been violated, addressed or satisfied. To locate requirements which have a certain status:

  • Locate the SEURAT Search Page from the Eclipse search window.
  • Select the Requirement radio button to indicate the search type.
  • Use the drop down list to select the Requirement Status which you wish to search for.
  • Click the Search button to execute the query.

The results of the search will be placed in the Eclipse search view. Each requirement will be listed in a table along with any arguments associated with that requirement. You may click on any of the column headers to sort the requirements based on that column's data. Double clicking on a row of the table will open the associated requirement in an editor which can be used to make modifications. Right clicking on any of the rows in the table will display a context menu containing additional actions you can perform.

The following actions are available from the context menu:
Open Requirement
Open an editor for the selected requirement or requirements.
Open Argument
Open an editor for the selected argument or arguments.
Open All Associated Rationale
Open an editor for each of the selected requirements and arguments.
