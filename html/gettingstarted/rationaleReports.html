Rationale Reports

SEURAT contains options for generating and exporting certain rationale reports. The Generate Traceability Matrix option can be found at the bottom of the Rationale Query Menu. The Export Rationale Tree option can be accessed by right-clicking on an element in the Rationale Tree.
  • Generate Requirements Traceability Matrix
  • Export Rationale Tree

While it is not considered a report, you may also be interested in the Graphical Rationale View.
